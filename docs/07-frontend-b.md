# 07 · B端设计（商户端 + 管理员端）

## 1. 整体布局（Admin Pro 风格）

```
┌─────────────────────────────────────────────────────────────────┐
│  [Logo] 中国移动积分商城管理系统          你好，张三 ▼  [退出]   │  ← TopBar
├──────────┬──────────────────────────────────────────────────────┤
│          │  首页 / 商品管理                                       │  ← 面包屑
│  导航菜单 ├──────────────────────────────────────────────────────┤
│  （竖排） │                                                       │
│          │                   内容区                              │
│          │                                                       │
│          │                                                       │
```

- 左侧菜单：深色背景（`#001529`），白色文字，选中条高亮绿色
- 菜单可折叠（展开/收起图标）

- 内容区：白色卡片，`padding: 20px`

## 2. 商户B端

### 访问控制
- `role=merchant` + `status=approved` 才能访问
- 状态为 `pending`（待审核）：登录后展示"审核中"页面，无法进入后台
- 状态为 `rejected`：展示拒绝原因，提示重新申请

### 左侧菜单

```
📊 数据概览
📦 商品管理
   ├─ 商品列表
   └─ 新增商品
🗄️ 库存管理
📋 订单管理
📈 数据分析
🎯 活动参与
```

## 5. 商户注册与审核状态机

```
[填写商户信息] → status=pending
      │
      ▼
[管理员审核中]
      │
   ┌──┴──┐
   ▼     ▼
通过     拒绝（附原因）
status=approved   status=rejected
   │                    │
   ▼                    ▼
可登录商户B端         展示拒绝原因
                   可修改后重新申请
```
