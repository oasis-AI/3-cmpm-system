## 2. 页面清单与交互规范

### 2.1 首页 `/`

#### Banner 轮播区（顶部）
- 3-5 张全宽 Banner，自动轮播（4s/张），支持左右箭头手动切换
- 点击 Banner 可跳转对应活动页

#### 快捷兑换区
两个并排卡片：

**话费兑换**
```
┌──────────────────────────────┐
│  📱 话费兑换                  │
│  [5元] [10元] [20元] [50元]  │
│  手机号：[___________]        │
│  需 500 积分                  │
│  [立即兑换]  当前积分: 1000   │
└──────────────────────────────┘
```
- 面值 Tab 切换，自动显示所需积分
- 未登录点击"立即兑换"弹出登录提示

**流量兑换**（同理，套餐为100M/500M/1G/5G）

#### 活动专区
- 横向滚动活动 Banner 列（4-6个活动）
- 每个活动卡片：Banner图 + 活动标题 + 结束倒计时

#### 商品分类快捷入口
- 10个分类图标（横排，两行），点击跳转对应分类的商品列表

#### 商城公告
- 右侧或顶部公告滚动条，最新3-5条，点击查看详情

#### 主推商品（Tab切换）
Tab：`精选` / `爆品` / `积分+现金` / `新品`
- 每行4个商品卡片，展示约20个
- 底部"查看更多"按钮跳转商品列表页，带当前 Tag 筛选

**商品卡片结构**：
```
┌────────────────┐
│   [商品主图]    │  ← 比例 1:1，hover 微放大
│ 🏷️ 甄选         │  ← 标签（自营/爆品/秒杀）
```

## 3. 路由守卫规则

```javascript
// 需要登录的路由
const authRequiredRoutes = ['/cart', '/checkout', '/user/*']

// 路由守卫
router.beforeEach((to, from, next) => {
  const authStore = useAuthStore()

  if (authRequiredRoutes.some(r => to.path.startsWith(r.replace('*', '')))) {
    if (!authStore.isLoggedIn) {
      next({ path: '/login', query: { redirect: to.fullPath } })
      return
    }
  }
  next()
})
```
